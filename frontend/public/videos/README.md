# Dossier Videos

Ce dossier est destiné à contenir les fichiers vidéo pour l'application.

## Fichier requis

- **background.mp4** : Vidéo de fond pour la section hero de la page d'accueil

## Instructions

1. Placez votre fichier vidéo `background.mp4` dans ce dossier
2. Format recommandé : MP4 (H.264)
3. Résolution recommandée : 1920x1080 ou supérieure
4. Taille recommandée : < 20 MB pour de meilleures performances

## Note

Si le fichier `background.mp4` n'est pas présent, l'application utilisera automatiquement un fond dégradé vert comme alternative.

## Alternative sans vidéo

Si vous ne souhaitez pas utiliser de vidéo, le CSS appliquera automatiquement un dégradé de couleur (vert) défini dans `App.css` pour la classe `.hero-video-wrapper`.
